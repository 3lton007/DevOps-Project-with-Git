## Technologies Used
- **CI/CD**: GitHub Actions

## Getting Started

### Prerequisites
- Docker 20.0+
- Node.js 16+


### Installation
```bash
# Clone repository
git clone https://github.com/3lton007/DevOps-Project-with-Git
cd DevOps=Project-with-Git

# Install dependencies
npm install

# Build Docker image
docker build -t devops-app .

# Run locally
docker-compose up
